# gene_expression_chatbot
A smart chatbot that helps you search gene expression data using natural language queries like:

“What’s the expression of TP53 in lung under infected condition?”

Built using Python, Streamlit, and MySQL, this project bridges the gap between biological data and user-friendly interfaces — designed especially for biotech students, bioinformaticians, and researchers who need quick access to expression data without complex SQL queries.

🚀 Features
💬 Chat Interface powered by Streamlit

🔍 Regex-based NLP to detect gene, tissue, and condition

🧠 Fetches results from MySQL database

📊 Interactive results displayed in a pandas DataFrame

🎨 Custom styled frontend with embedded HTML/CSS

📚 Technologies Used-
Python

Streamlit

MySQL

pandas

Regex (re library)

🧪 How to Use-
pip install -r requirements.txt
streamlit run FinalProject.py
